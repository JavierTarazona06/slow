# Archived Source Variants

These files were moved out of `src/slow/` because they are not imported by the active application.

- `main_deteccion.py` was a duplicate of the active `src/slow/MAIN_DETECCION.py`.
- `MAIN_DETECCION2.py` was an alternate detector variant.
- `Funciones.py` was an empty placeholder class.

They are preserved here for reference while keeping the active package easier to maintain.
